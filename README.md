## 📌 Project Overview

**IronEdge Fitness** is a fully responsive, professional gym website built for a fitness studio based in Hyderabad, Telangana. This project was developed as part of the **Future Intern Internship – Task 3**, which challenged interns to build a real-world business website and pitch it to a local business owner.

---

## 🎯 The Pitch

> *"Most local gyms in Hyderabad rely only on Instagram and word-of-mouth. A professional website builds credibility, attracts new members around the clock, showcases plans and trainers, and converts visitors into paying customers."*

---

## ✨ Features

- ⚡ **Animated Hero Section** — Live counters (Members, Classes, Trainers, Years)
- 🏋️ **Services / Programs** — Strength, HIIT, CrossFit, Personal Training, Yoga, Nutrition
- 👨‍💼 **Trainer Profiles** — Specializations and bios
- 💳 **Membership Plans** — Starter (₹999), Pro (₹1,999), Elite (₹3,499)
- 📸 **Transformation Gallery** — Before/after style image grid
- ⭐ **Testimonials** — Auto-scrolling carousel with member reviews
- 📅 **Weekly Class Schedule** — Color-coded timetable (Mon–Sat)
- 📬 **Contact Form** — With success toast notification
- 📍 **Location Section** — Address, phone, email, and hours
- 📱 **Fully Responsive** — Mobile-first design
